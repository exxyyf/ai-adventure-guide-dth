## Summary

TripGenie генерирует маршруты и даёт ответы в приложении — в том числе по локальным особенностям и культурным активностям. Подходит для быстрого получения советов «как себя вести/какие фестивали не пропустить». Можно бронировать
## Functions

персонализированные итинерарии, советы по достопримечательностям и локальным обычаям, голосовые и текстовые ответы внутри приложения.
## Technologies

LLM + интеграция внутр. API бронирования и POI; retrieval для проверки данных
